Gemini投資レポートシステム開発計画書 - Gemini.md
1. システム概要
本システムは、無料の公開情報ソースのみを統合し、Gemini APIを用いて個人向け週次投資レポートを自動生成する分析エンジンです。レポートは、マクロ経済、ミクロ経済、投資家心理といった多層的な視点から、市場の複合的な要因を分析します。

情報ソース（入力）:

マクロ経済: @AC_Investor (YouTube)、国際通貨基金（IMF）、世界銀行、BIS（国際決済銀行）

ミクロ経済/トレンド: NewsPicks（無料記事）、Fred Wilson (AVC)

定性情報/心理: 中嶋聡さんのメルマガ、金融情報サイト（空売り情報など）、Twitter（特定アカウント）

客観的指標: VIX指数、主要株価指数（日経平均、S&P 500、NASDAQ）、主要暗号資産価格、為替レート（ドル円）

レポート内容（出力）:

市場の複合的な要因分析と、各市場の個別動向。

投資家心理の多角的分析。

次なるトレンドの深掘り。

今後の展望と注視すべきポイント。

2. 実装案と技術スタック
本システムは、Pythonで開発します。PythonはAPI連携のライブラリが豊富で、開発が迅速に進められます。

モジュール名	役割	使用技術/ライブラリ
データ収集モジュール	各情報ソースからデータを取得。ウェブスクレイピングや公開APIを利用。	Python requests, BeautifulSoup, youtube-dl, yfinance, pandas, snscrape
Gemini API連携モジュール	収集したデータをAPIに渡し、高度な分析・レポート生成を行う。	Python google-generativeai ライブラリ
レポート生成モジュール	Gemini APIの出力をMarkdown形式のレポートに整形・出力する。	Python os, markdown
スケジューリング	週に一度のレポート生成を自動化。	cron (Linux) / タスクスケジューラ (Windows)

3. 開発手順
ステップ1: 環境構築とAPIキー取得
Python 3.8+ 環境をセットアップします。

必要なライブラリをインストールします。

Bash

pip install google-generativeai requests beautifulsoup4 pandas youtube-dl yfinance snscrape pandas-ta
Gemini APIキーを取得し、環境変数に設定します。

Bash

export GEMINI_API_KEY='YOUR_API_KEY'
ステップ2: データ収集モジュール実装
各情報ソースの特性に合わせて、データ収集スクリプトを構築します。

YouTube: youtube-dl を用いて、@AC_Investorチャンネルの最新動画の文字起こしテキストを取得します。

ニュースレター/ブログ: 公開されている無料記事をターゲットに、ウェブスクレイピング（requestsとBeautifulSoup）を行い、テキストデータを抽出します。

公的機関レポート: IMF、世界銀行、BISの公開ウェブページから、最新レポートの要約部分や本文をスクレイピングで取得します。

株価・指標データ: yfinanceなどの無料ライブラリを利用し、日次データを取得・保存します。

Twitter: snscrapeを利用し、指定したアカウントの過去1週間のツイートを取得し、outputディレクトリに保存します。

ステップ3: Gemini APIプロンプト設計（最も重要）
レポートの多層的な分析を可能にするため、複数の情報源を統合し、論理的推論を促す複雑なプロンプトを作成します。

プロンプトの構成例
Python

# Gemini APIへ送るプロンプトの例

prompt = """
以下の多様な情報源を統合し、グローバル市場のマクロ・ミクロ経済、投資家心理、そして次のトレンドに関する週次投資レポートを、プロフェッショナルなトーンで作成してください。レポートはMarkdown形式で、具体的な売買指示を含まず、あくまで情報提供に徹してください。

### 入力データ

- **マクロ経済（@AC_Investor, IMF, 世界銀行, BIS）**:
  - @AC_Investor動画の文字起こし: {ac_investor_text}
  - IMF/世界銀行/BISの最新レポート要約: {public_report_text}
- **トレンド/ミクロ経済（NewsPicks, AVC）**:
  - NewsPicksの注目無料記事要約: {newspicks_text}
  - Fred Wilsonのブログ（AVC）記事要約: {avc_text}
- **定性分析/投資家心理（中嶋聡, 公開情報, Twitter）**:
  - 中嶋聡さんメルマガの要約: {nakajima_text}
  - Twitterの注目アカウントのツイート: {twitter_text}
  - 一般公開されている空売り残高情報、マーケットセンチメント指標: {public_sentiment_data}
- **客観的指標**:
  - VIX指数: {vix_value}
  - 主要市場指数と為替データ: {index_data}

### レポートの構成

#### 1. 市場全体のマクロ経済分析

* **動画と公的機関レポートの統合分析**:
    * グローバル経済の主要トレンドを要約。
    * @AC_Investorの分析に、IMFやBISのレポートで示される公的な見解を加え、より信頼性の高いマクロ分析を提示してください。

#### 2. 投資家心理の多角的分析

* **VIX指数と公開情報の統合**:
    * VIX指数の動きから表面的な市場の恐怖度を評価。
    * 一般公開されている空売り残高データから、市場の隠れた弱気センチメントを分析してください。
* **専門家の見解**:
    * 中嶋聡さんのメルマガを解析し、市場の今後の方向性に対する専門家の見解を論じてください。

#### 3. 次なるトレンドの深掘り

* **NewsPicksとAVCのクロス分析**:
    * NewsPicksで話題になっている日本の特定のスタートアップや産業トレンドを、Fred Wilson氏のブログで示されるテクノロジーの未来に関する見解と関連付けて論じてください。

#### 4. 市場別展望と結論

* **各市場の個別分析**:
    * 日本株式、米国株式、暗号資産の各市場を個別に分析し、上記で統合したマクロ・ミクロ・心理的な要因がそれぞれにどのような影響を与えているかを論じてください。
* **今週の最重要ポイントTOP3**:
    * レポート全体から、最も重要な洞察を3つに絞り、箇条書きで提示してください。
* **次週への展望**:
    * 来週の市場で特に注視すべきイベントやテーマを簡潔にまとめ、レポートを締めくくってください。
"""

### ステップ4: レポート生成と出力

Gemini APIからのレスポンスをMarkdownファイルとして保存するスクリプトを作成します。

### ステップ5: スケジューリング

`cron`（Linux）やタスクスケジューラ（Windows）を用いて、毎週決まった曜日・時間にスクリプトが自動実行されるように設定します。
